Concurrently

The concurrently is a utility which allows you to

"Run multiple commands concurrently"

concurrently readme.

What else to add? It's a feature-rich utility that boosts your development process! Though, in this article, I will show you the very basics of its usage.

Task

I have decided to use this utility because I want to display the actual source code of this project.

But, I want any updates of the code to be added automatically without my involvement.

To achieve this goal, I wrote a node.js script, which is running concurrently with angular scripts.

Solution

First of all, add the concurrently utility to the project:

yarn add -D concurrently

Next, modify the package.json. In my case, it looks like this:

"start": "concurrently -k \"ng serve\" \"nodemon sources.js\"",

What it does:

-k
kill other processes if one exits or dies.
ng serve
starts angular development server.
nodemon sources.js
runs my script upon files update.

The output in the console should look like this:

{{ outputCode }}

Note, that each task output is prefixed with the task id, e.g. [0].

Thus, you can understand which thread generated which line in the output for later analysis.

Conclusion

This utility is extremely useful if you need to run several tasks simultaneously.

This example demonstrates only the tip of the iceberg. This utility can do much more and thus can boost your productivity.
