The list of required tools

Here goes the list of tools which one will need to have to start developing.

Please, note that the order of the list is also important.

For example, Yarn will not work without NPM, etc.

Ok let's get started

Node.js

The Node.js provides a powerful V8 JavaScript engine, a package manager (aka NPM), server-side JavaScript runtime environment, etc.

Yarn

The Yarn package manager is a superset for an NPM.

Why use Yarn?

First of all, it is much faster.

Secondly, Yarn will recreate all of the dependencies you use in your project with the same version on any remote workstation.

IDE

You may use any IDE which fits you the most.

Here are some of them Eclipse, NetBeans, WebStorm, Visual Studio Code.

If you are a hardcore type of developer you may also use Atom, Sublime Text or even Vim.

I'm using Visual Studio Code if you are using it too then checkout list of extensions I'm using daily to be more productive.

Git

Git is a powerful yet flexible, still easy to use version control system.

You can use different software to sync your work between local and remote repositories and other developers.

Though, the official package of git includes such tools as gitk and git-gui you may also use any of programs listed on the official site.

If you prefer to use the CLI, then take a look at the list of commands which I'm using daily.

Angular

To start a new Angular project follow these steps.

First, install an Angular CLI via NPM.

npm install -g @angular/cli

Or using Yarn:

yarn add global @angular/cli

Second, create a new project:

ng new app

Before the creation, Angular will ask you about the basic features of the new project.

The first question is about enabling Angular routing in your project (default option is no).

The second question is about CSS handling (default option is pure CSS).

After all of the required operations are done, go to the project folder.

cd app

Finally, run the project!

ng serve

Optionally you can provide --open flag, which will open your default browser once the project is ready to start.

ng serve --open

To get a list of available commands run the following:

ng help
Bootstrap

To build applications fast, one should use CSS component library.

My favorite is Bootstrap!

So, to add this component library to your project start CLI anywhere inside your project's folder and install it via NPM:

npm install bootstrap

Or via Yarn:

yarn add bootstrap
Font Awesome

One more thing you can to do is to install the Font Awesome, though this step is optional.

This toolkit will allow you to add awesome SVG icons into your project.

Which, in turn, will enrich the visual experience and will add a neat look to your project.

Anywhere inside your project's folder run:

npm install @fortawesome/fontawesome-svg-core @fortawesome/free-brands-svg-icons @fortawesome/free-regular-svg-icons @fortawesome/free-solid-svg-icons @fortawesome/angular-fontawesome

Or via Yarn:

yarn add @fortawesome/fontawesome-svg-core @fortawesome/free-brands-svg-icons @fortawesome/free-regular-svg-icons @fortawesome/free-solid-svg-icons @fortawesome/angular-fontawesome

Here goes short description for each package installed:

@fortawesome/fontawesome-svg-core - the toolkit itself.

@fortawesome/free-brands-svg-icons - the set of free branded icons, such as Google, Twitter, Facebook, etc.

@fortawesome/free-regular-svg-icons - the set of free regular icons.

@fortawesome/free-solid-svg-icons - the set of free solid icons.

@fortawesome/angular-fontawesome - this is a wrapper library for use in Angular projects.
