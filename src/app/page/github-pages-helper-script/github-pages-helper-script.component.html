A helper script for the Github Pages.

When I have started this project, I have found that Github Pages are requiring to have a copy of an index.html file named 404.html.

There are projects which are solving this problem. For example, there is an angular-cli-ghpages project.

But I wanted to solve the problem personally. Plus, I didn't want to add an extra layer to the project.

As one of my teachers said:

"Follow the KISS rule."

That is why I had created this script: it is simple yet robust.
