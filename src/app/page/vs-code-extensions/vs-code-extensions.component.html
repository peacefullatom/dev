Visual Studio Code extensions which help you to be more productive

Every day I'm writing a lot of JavaScript code. And also creating HTML templates and defining styles for them.

So, this means that I have a lot of routine tasks which are eating my time, but I can't avoid them.

But, I'm using a Visual Studio Code! And this IDE has a lot of extensions which are making my life easier.

TSLint

A real-time linter for TypeScript files.

First of all, this extension will highlight errors. Moreover, TSLint will show the possible fixes list.

This extension also has an auto-fix feature for most common mistakes.

TypeScript Hero

This extension is a real life-saver because it can sort imports and remove unused imports, etc.

And thus enables you to have pretty and clean code: no more "spaghetti" imports - life is beautiful!

Prettier

The only thing this extension can't handle is to organize imports.

All other tasks performed with brilliant ease and speed, and this is awesome.

Any messy file will become human-readable in a matter of fractions of seconds once you save it.

Also, this extension supports a lot of file types and able to format not only code.

On the list of supported types where is Python, Java, C#, CSS, LESS, JSON, etc.

Visual Studio IntelliCode

As the first paragraph of the description of this extension says:

The Visual Studio IntelliCode extension provides AI-assisted development features for Python, TypeScript/JavaScript and Java developers in Visual Studio Code, with insights based on understanding your code context combined with machine learning.

What else to say? This AI-enabled extension is boosting up your productivity up to the sky!

The IntelliCode extension allows you to use your code at its best at the greatest rate.
