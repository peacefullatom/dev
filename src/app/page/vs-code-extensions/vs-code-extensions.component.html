Visual Studio Code extensions which help you to be more productive

Every day I'm writing a lot of JavaScript code. And also creating HTML templates and defining styles for them.

So, this means that I have a lot of routine tasks which are eating my time, but I can't avoid them.

But, I'm using a Visual Studio Code! And this IDE has a lot of extensions which are making my life easier.

TSLint

A real-time linter for TypeScript files.

First of all, this extension will highlight errors. Moreover, TSLint will show the possible fixes list.

This extension also has an auto-fix feature for most common mistakes.

TypeScript Hero

This extension is a real life-saver because it can sort imports and remove unused imports, etc.

And thus enables you to have pretty and clean code: no more "spaghetti" imports - life is beautiful!

Prettier

The only thing this extension can't handle is to organize imports.

All other tasks performed with brilliant ease and speed, and this is awesome.

Any messy file will become human-readable in a matter of fractions of seconds once you save it.

Also, this extension supports a lot of file types and able to format not only code.

On the list of supported types where is Python, Java, C#, CSS, LESS, JSON, etc.

Visual Studio IntelliCode

As the first paragraph of the description of this extension says:

The Visual Studio IntelliCode extension provides AI-assisted development features for Python, TypeScript/JavaScript and Java developers in Visual Studio Code, with insights based on understanding your code context combined with machine learning.

What else to say? This AI-enabled extension is boosting up your productivity up to the sky!

The IntelliCode extension allows you to use your code at its best at the greatest rate.

GitLens

A GitLens extension covers all types of tasks connected with version control.

This extension allows you to browse branches, contributors, stashes, and tags.

Also, it enables you to navigate and compare a file and a line history.

You can perform a comparison of commits by branch, tag, or ref.

And even search commits by message, author, id, file, etc.

The file blame annotations mode helps you to visualize code authorship.

All this power is available inside an IDE.

IntelliSense for CSS class names in HTML

Let's say you are using some external CSS library in your project, or you are new to some project which has a rich CSS.

In any case, you should memorize a lot of new class names.

To simplify the process, you should use this extension.

This utility will analyze and cache all CSS definitions available.

After that, you will have the ability to see the class names inside HTML files during editing.

Debugger for Chrome

If you want to debug your favorite app in Chrome and have features of IDE, then this extension is for you.

All you need is configure your workspace and Chrome browser.

After that set breakpoints and start debugging mode.

Bracket Pair Colorizer

It is not always an easy task to track a matching pair.

This extension enables colorizing for matching pairs.

And thus you can always understand inside of which scope you currently are.

Spell Checker

This feature-rich extension will help you avoid typos in your code, comments, etc.

It supports several languages and a variety of file types.

Also, this extension understands different naming conventions, such as camelCase, snake_case, etc.

Another noticeable feature is a suggestion list for misspelled words.

Settings Sync

If you are using Code on several workstations, then you will need this extension.

It allows you to sync your settings between different instances of IDE.

These settings are including hotkeys, themes, extensions, snippets, etc.
