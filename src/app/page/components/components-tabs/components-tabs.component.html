Tabs

Tabs are used everywhere. They are helping to divide the user interface into logical parts.

This approach helps the user to navigate through the interface. On the other hand, this also helps the developer to develop and maintain the project.

This article will show a basic way to add the tabs component to the project.

Description

The component should allow the user to navigate through internal links (.e.g. components/intro). Also, it should handle a custom click events.

Component

The tabs component takes an array of links descriptions.

The behavior of the tabs is limited to link-like and button-like styles. By default, all tabs are interpreted as buttons.

Additionally, there is an option to render tabs with equal width if possible.
